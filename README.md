# CS50 – Credit

## Description

This project revisits the _Credit_ exercise from CS50 by implementing the same credit card validator in Python instead of C.

## How it works

The program:

- Prompts the user for a credit card number
- Validates the number using Luhn's algorithm
- Identifies the card issuer based on its length and starting digits
- Prints the card type or "INVALID"

## Usage

```
python credit.py
```

## Example

```
Number: 378282246310005
AMEX
```
